## Learning Map For A Claims Intern

| Layer | What You Need To Understand | Claims Role Focus |
|---|---|---|
| Product layer | 险种、保险责任、责任免除、等待期、免赔额、比例、限额 | 判断是否可能赔 |
| Policy layer | 投保、核保、承保、回执、签收日、生效日、犹豫期、保全 | 判断合同状态是否有效 |
| Event layer | 出险原因、事故时间、诊断、治疗、费用 | 判断事故是否进入责任范围 |
| Evidence layer | 申请书、身份证、病历、诊断证明、发票、清单、医保结算单 | 判断材料是否完整可信 |
| Calculation layer | 已报销金额、免赔额、赔付比例、日限额、年限额、剩余额度 | 算应赔金额 |
| Decision layer | 正常赔付、补件、照会、复核、拒付、协议处理 | 形成理赔结论 |

## Your Current Study Stage

你现在已经进入“审核”部分，说明前面的保单生命周期和立案流程已经开始串起来。接下来学习重点建议放在：

1. 医疗险条款如何读。
2. 费用补偿型医疗险如何理算。
3. 常见责任免除如何识别。
4. 病历、发票、费用清单、医保结算单之间如何互相印证。
5. 哪些案件需要补件、照会、复核或升级。
