## 双核学习地图

| 层级 | 我要理解什么 | 双核岗位重点 |
|---|---|---|
| 产品层 | 险种、保险责任、责任免除、等待期、免赔额、比例、限额 | 核保看能不能承保，核赔看是否可能赔 |
| 保单层 | 投保、核保、承保、回执、签收日、生效日、犹豫期、保全 | 判断合同状态和责任是否有效 |
| 事故层 | 出险原因、事故时间、诊断、治疗、费用 | 判断事故是否进入责任范围 |
| 证据层 | 申请书、身份证、病历、诊断证明、发票、清单、医保结算单 | 判断材料是否完整可信 |
| 理算层 | 已报销金额、免赔额、赔付比例、日限额、年限额、剩余额度 | 算应赔金额 |
| 结论层 | 正常赔付、补件、照会、复核、拒付、协议处理 | 形成能解释、能复核的结论 |

## 我当前的学习重点

我现在要把核保和核赔连起来看：核保决定风险怎么进来，核赔决定事故发生后怎么按合同处理。接下来学习重点放在：

1. 医疗险条款如何读，尤其是医院范围、等待期、免责和费用补偿。
2. 核保健康告知如何影响后续理赔，尤其是既往症和短期出险。
3. 费用补偿型医疗险如何理算。
4. 病历、发票、费用清单、医保结算单之间如何互相印证。
5. 哪些案件需要补件、照会、复核或升级。
