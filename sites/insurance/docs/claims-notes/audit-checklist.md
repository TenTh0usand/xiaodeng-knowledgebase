## 1. 保单有效性

| 检查项 | 问题 |
|---|---|
| 保单状态 | 保单在出险日是否有效？ |
| 缴费状态 | 是否欠费？是否在宽限期？是否失效或复效？ |
| 主附险 | 本次申请对应主险还是附加险？责任是否仍有效？ |
| 等待期 | 疾病出险是否在等待期后？复效或重新投保是否重新计算等待期？ |
| 特别约定 | 是否有除外、加费、特别约定或职业限制？ |

## 2. 申请人和身份

| 检查项 | 问题 |
|---|---|
| 申请人 | 申请人是否有资格申请？ |
| 被保险人 | 就医人是否为被保险人本人？ |
| 受益人 | 医疗金受益人通常是否为被保险人本人？条款有无特别约定？ |
| 授权委托 | 委托他人办理或领取时，授权委托是否完整？ |
| 收款账户 | 收款账户是否与申请人/受益人匹配？ |

## 3. 事故和诊断

| 检查项 | 问题 |
|---|---|
| 原因 | 是疾病、意外，还是其他原因？ |
| 日期 | 首次确诊日、入院日、出院日、事故日是否清楚？ |
| 诊断一致性 | 诊断证明、病历、检查报告是否一致？ |
| 既往症 | 是否可能属于既往症或投保前已存在症状？ |
| 免责 | 是否涉及先天性疾病、遗传性疾病、康复疗养、美容牙科、酒驾等免责？ |

## 4. 医院和治疗

| 检查项 | 问题 |
|---|---|
| 医院等级 | 医院是否符合条款定义？是否为二级及以上或指定医院？ |
| 入出院事实 | 是否正式办理入出院？是否存在门诊观察、家庭病床或挂床风险？ |
| 治疗必要性 | 治疗是否与诊断匹配，是否合理且医学必要？ |
| 特殊科室 | 特需、国际部、私立医院、康复科是否在责任范围内？ |
| 同一次住院 | 再次入院是否与前次住院原因相同，是否应视为同一次住院？ |

## 5. 材料

| 材料 | 为什么重要 |
|---|---|
| 理赔申请书 | 明确申请责任、申请人、事故和收款信息。 |
| 身份证件 | 核验投保人、被保险人、受益人或委托人身份。 |
| 诊断证明 | 确认疾病或事故结论。 |
| 病历资料 | 还原治疗过程、既往病史、主诉和诊疗依据。 |
| 入出院记录 | 判断住院事实、住院天数、入出院诊断。 |
| 发票 | 金额凭证，判断是否原件、是否已报销。 |
| 费用清单 | 拆分药品、检查、治疗、床位、材料等费用。 |
| 医保结算单 | 判断医保已报销金额和个人自付部分。 |
| 病理/检验/影像报告 | 支撑重疾、肿瘤、手术或复杂诊断。 |
| 其他补偿证明 | 避免超过实际损失重复赔付。 |

## 6. 理算

| 项目 | 怎么想 |
|---|---|
| 可赔费用 | 先从总费用中剔除不属于责任或不合理的费用。 |
| 其他补偿 | 扣除医保、公费医疗、第三方责任方、其他商保等已补偿金额。 |
| 免赔额 | 按年度、按次或按责任扣除免赔额。 |
| 赔付比例 | 根据是否经医保结算、医院类型、责任类型套用比例。 |
| 分项限额 | 注意床位费、门急诊、特药、住院前后门急诊等分项限额。 |
| 年度限额 | 检查年度限额和剩余额度。 |
| 日限额 | 住院津贴或特定责任可能按天数和日额计算。 |
| 欠费或贷款 | 条款允许时，赔付时可能扣除欠交保费或保单贷款。 |

## 7. 异常信号

| 异常信号 | 后续动作 |
|---|---|
| 出险时间接近生效日或等待期结束日 | 查投保前病史、首诊记录、体检记录。 |
| 住院天数异常长或与病情不匹配 | 查医嘱、病程记录、出院标准。 |
| 费用结构异常 | 查费用明细、药品和治疗必要性。 |
| 多次小额重复理赔 | 查同一事故、同一次住院、重复票据。 |
| 发票或清单信息不一致 | 要求补件或联系医院核实。 |
| 投保后短期高额理赔 | 查健康告知、既往症、核保记录。 |
| 代理人或第三方代办异常积极 | 核验客户授权和材料真实性。 |

## 8. 结论怎么写

理赔结论要写清楚三件事：

1. **事实**：谁在什么时间因什么原因就医或出险。
2. **依据**：对应哪条保险责任、免责、等待期、免赔额、比例或限额。
3. **结果**：赔付金额、拒付原因、补件要求或复核意见。

## 9. 简短审核模板

```text
案件事实：
被保险人于 YYYY-MM-DD 因 [疾病/意外原因] 在 [医院] 就诊/住院，申请 [责任名称] 理赔。

合同状态：
保单于 YYYY-MM-DD 生效，事故发生时保单 [有效/宽限期/失效/复效后]，等待期 [已过/未过/不适用]。

责任判断：
本次事故 [属于/不属于] 条款约定的 [保险责任]，未见/存在 [责任免除或特别约定]。

材料审核：
已提供 [诊断证明、病历、发票、费用清单、医保结算单等]，材料 [完整/需补充]。

理算结果：
合理医疗费用 X 元，扣除其他补偿 X 元，扣除免赔额 X 元，按 X% 给付，受限额 X 元约束，本次应赔 X 元。
```

## 10. 客户、公司、我的视角

| 视角 | 医疗险审核怎么理解 |
|---|---|
| 客户 | 客户关心的是能不能报、多久报、为什么有些费用报不了。解释时要说人话，但不能离开条款。 |
| 公司 | 医疗险赔付频率高，审核要防重复赔付、超责任赔付、不合理医疗和欺诈风险。 |
| 我 | 每个案件都按“保单有效性、责任范围、材料证据、费用理算、异常信号”走一遍，别一上来就算钱。 |
