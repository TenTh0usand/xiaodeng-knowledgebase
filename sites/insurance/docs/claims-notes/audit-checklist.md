# Medical Claim Audit Checklist

这一页是给“审核阶段”准备的工作清单。你今天学到审核部分后，可以用它来对照每个医疗险案件。

## 1. Policy Validity

| Check | Question |
|---|---|
| Policy status | 保单在出险日是否有效？ |
| Premium status | 是否欠费？是否在宽限期？是否失效或复效？ |
| Product and rider | 本次申请对应主险还是附加险？责任是否仍有效？ |
| Waiting period | 疾病出险是否在等待期后？复效或重新投保是否重新计算等待期？ |
| Special agreement | 是否有除外、加费、特别约定或职业限制？ |

## 2. Claimant And Identity

| Check | Question |
|---|---|
| Applicant | 申请人是否有资格申请？ |
| Insured | 就医人是否为被保险人本人？ |
| Beneficiary | 医疗金受益人通常是否为被保险人本人？条款有无特别约定？ |
| Authorization | 委托他人办理或领取时，授权委托是否完整？ |
| Bank account | 收款账户是否与申请人/受益人匹配？ |

## 3. Accident And Diagnosis

| Check | Question |
|---|---|
| Cause | 是疾病、意外，还是其他原因？ |
| Date | 首次确诊日、入院日、出院日、事故日是否清楚？ |
| Diagnosis consistency | 诊断证明、病历、检查报告是否一致？ |
| Pre-existing condition | 是否可能属于既往症或投保前已存在症状？ |
| Exclusion | 是否涉及先天性疾病、遗传性疾病、康复疗养、美容牙科、酒驾等免责？ |

## 4. Hospital And Treatment

| Check | Question |
|---|---|
| Hospital level | 医院是否符合条款定义？是否为二级及以上或指定医院？ |
| Admission | 是否正式办理入出院？是否存在门诊观察、家庭病床或挂床风险？ |
| Treatment necessity | 治疗是否与诊断匹配，是否合理且医学必要？ |
| Special department | 特需、国际部、私立医院、康复科是否在责任范围内？ |
| Same hospitalization | 再次入院是否与前次住院原因相同，是否应视为同一次住院？ |

## 5. Documents

| Document | Why It Matters |
|---|---|
| Claim application form | 明确申请责任、申请人、事故和收款信息。 |
| ID documents | 核验投保人、被保险人、受益人或委托人身份。 |
| Diagnosis certificate | 确认疾病或事故结论。 |
| Medical records | 还原治疗过程、既往病史、主诉和诊疗依据。 |
| Admission and discharge records | 判断住院事实、住院天数、入出院诊断。 |
| Invoice | 金额凭证，判断是否原件、是否已报销。 |
| Expense list | 拆分药品、检查、治疗、床位、材料等费用。 |
| Social medical insurance settlement sheet | 判断医保已报销金额和个人自付部分。 |
| Pathology / lab / imaging reports | 支撑重疾、肿瘤、手术或复杂诊断。 |
| Other compensation proof | 避免超过实际损失重复赔付。 |

## 6. Calculation

| Item | How To Think |
|---|---|
| Eligible expense | 先从总费用中剔除不属于责任或不合理的费用。 |
| Other compensation | 扣除医保、公费医疗、第三方责任方、其他商保等已补偿金额。 |
| Deductible | 按年度、按次或按责任扣除免赔额。 |
| Reimbursement ratio | 根据是否经医保结算、医院类型、责任类型套用比例。 |
| Sub-limit | 注意床位费、门急诊、特药、住院前后门急诊等分项限额。 |
| Annual limit | 检查年度限额和剩余额度。 |
| Daily limit | 住院津贴或特定责任可能按天数和日额计算。 |
| Outstanding premium or loan | 条款允许时，赔付时可能扣除欠交保费或保单贷款。 |

## 7. Red Flags

| Red Flag | Follow-Up |
|---|---|
| 出险时间接近生效日或等待期结束日 | 查投保前病史、首诊记录、体检记录。 |
| 住院天数异常长或与病情不匹配 | 查医嘱、病程记录、出院标准。 |
| 费用结构异常 | 查费用明细、药品和治疗必要性。 |
| 多次小额重复理赔 | 查同一事故、同一次住院、重复票据。 |
| 发票或清单信息不一致 | 要求补件或联系医院核实。 |
| 投保后短期高额理赔 | 查健康告知、既往症、核保记录。 |
| 代理人或第三方代办异常积极 | 核验客户授权和材料真实性。 |

## 8. Decision Writing

理赔结论要写清楚三件事：

1. **事实**：谁在什么时间因什么原因就医或出险。
2. **依据**：对应哪条保险责任、免责、等待期、免赔额、比例或限额。
3. **结果**：赔付金额、拒付原因、补件要求或复核意见。

## 9. A Short Audit Template

```text
案件事实：
被保险人于 YYYY-MM-DD 因 [疾病/意外原因] 在 [医院] 就诊/住院，申请 [责任名称] 理赔。

合同状态：
保单于 YYYY-MM-DD 生效，事故发生时保单 [有效/宽限期/失效/复效后]，等待期 [已过/未过/不适用]。

责任判断：
本次事故 [属于/不属于] 条款约定的 [保险责任]，未见/存在 [责任免除或特别约定]。

材料审核：
已提供 [诊断证明、病历、发票、费用清单、医保结算单等]，材料 [完整/需补充]。

理算结果：
合理医疗费用 X 元，扣除其他补偿 X 元，扣除免赔额 X 元，按 X% 给付，受限额 X 元约束，本次应赔 X 元。
```
